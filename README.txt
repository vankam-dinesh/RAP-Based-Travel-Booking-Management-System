# RAP-Based Travel Booking Management System

## Description
A modern SAP RAP application built using CDS Views, Managed Behavior, OData V4, and Fiori Elements UI.

## Features
- Create, Update, Delete Travel Bookings
- CDS-based data model
- RAP Managed Scenario
- Fiori Elements List Report & Object Page

## Technologies
- ABAP RAP
- CDS Views
- OData V4
- SAP Clean Core
